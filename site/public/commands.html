Commands

Slash commands pibarm registers in pi. Type them in the pi editor; arguments in <angle brackets> are required, [square brackets] optional.

Planning & execution

CommandWhat it does
/plan <task>Enter read-only plan mode and ask for a plan. Edits are disabled and bash is restricted while planning.
/plan-modeToggle read-only plan mode manually.
/plan-showShow the last captured plan, its status, and parsed steps.
/approve-plan [active|worktree <name>]Approve and execute the captured plan — in the active checkout or an isolated worktree.
/refine-plan <feedback>Revise the captured plan; approval is required again afterwards.
/execute-planExecute the last captured plan in the active checkout.
/execute-plan worktree <name>Execute the plan in a new repo-local git worktree (.pi/wt/<name>, branch pibarm/<name>).

Worktrees

CommandWhat it does
/worktreesList git worktrees for this repo.
/worktree-diff <path>Show status and diff stat for a worktree.
/worktree-remove [--force] <path>Remove a worktree after reviewing, merging, or abandoning it.

Matrix (WezTerm agent panes)

Visible, parent-controlled agents in WezTerm splits/tabs. Requires the wezterm CLI. See the Matrix demo.

CommandWhat it does
/matrix-helpExplain when/how to use Matrix and its prior art.
/matrix <task>Start a Matrix with scout + planner panes for a task.
/matrix-attachOpen or focus the session-specific Matrix workspace window.
/matrix-spawn <role> <task>Spawn one agent (scout / planner / worker / reviewer) in a pane.
/matrix-send <role> <message>Send a message to a still-running Matrix pane.
/matrix-capture [role]Capture recent output from Matrix panes/logs.
/matrix-join [role|all]Wait for agents to finish, capture logs, and clean up panes.
/matrix-listList Matrix agents and panes for this session.
/matrix-kill [role|all]Kill Matrix panes.

Tasks & watchers

CommandWhat it does
/tasksShow all todo and delegated-agent items from the shared task widget in an expanded view.
/watchersList active watcher sibling agents.
/watcher-stop [name]Stop one or all watcher sibling agents.

Repo, forge & review

GitHub via gh, SourceHut via hut — local CLI auth only, no tokens stored.

CommandWhat it does
/repo-statusShow git/forge/CI status and update the pi statusline.
/forge [github|sourcehut|auto]Show, set, or reset the remembered forge for this repo (stored in ignored .pi/forge.json).
/forge-prsList PRs/patches using the detected or configured forge.
/forge-ciList CI runs/builds.
/forge-ticketsList issues/tickets.
/review [#number|url|branch]Start a PR/patch review; GitHub findings are posted inline when possible.

Presets, MCP & Obsidian

CommandWhat it does
/presetList role presets configured in .pi/agent-presets.json.
/preset <name>Apply a preset — provider/model, thinking level, active tools (e.g. planner, executor).
/mcporterShow configured mcporter command templates.
/mcporter <args...>Run raw mcporter args from inside pi.
/obsidian-statusShow Obsidian export settings resolved from pi settings.
/obsidian-exportExport the current session to the configured Obsidian vault.

Inline shell

Type !<command> as a prompt to run a local shell command immediately and show the output in the transcript, without invoking the model. !! is left to pi's built-in bash handling.

!git status --short
